NEC NL4864HL11-01B Specifications

Display: 3.5" (8.9cm) LTPS TFT LCD
Aspect Ratio: 3:4
Pixel Format: 480 x 640 (H x V)
Brightness: 200 cd/m2
Contrast Ratio: 180:1
Response Time: 30ms
Dimensions: 63.5 x 85.0 x 4.2mm (W x H x D)
Weight: 43g

NEC's NL4864HL11-01B is a 3.5" TFT LCD module that sports a very high pixel format of 480 x 640 and is geared for mobile applications. The high pixel format is achieved by using the Low Temperature Poly-Silicon (LTPS) process. Many of the driver electronics are integrated into the panel as well via System on Glass (SOG) technology. The aspect ratio is 3:4 (or 4:3 if in landscape orientation) and is probably not an ideal solution for many smartphone displays as those are shifting strongly toward a wide aspect ratio. It would be only time for NEC to develop a 3.5" LTPS SOG LCD module that sports a 800 x 480 pixel format. Fingers crossed.

Source: NEC
