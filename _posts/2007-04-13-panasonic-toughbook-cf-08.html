Panasonic ToughBook CF-08

  • Display Size: 10.4"
  • Aspect Ratio: 4:3
  • Pixel Format: 1024x768
  • Form Factor: Tablet PC (Slab)
  • Operating System: Windows CE 5.0
  • Touch: Yes
  • CPU: 312MHz PXA270
  • RAM: 64MB RAM, 64MB ROM
  • Connectivity: Ethernet, 802.11b/g, Bluetooth 1.2
  • Battery Life: 38 watt-hour Lithium Ion battery lasts 4.5 - 14 hours
  • Dimension: 267 x 208 x 36mm
  • Weight: 1.2kg (2.65lbs)


Rugged PC Review: The Panasonic ToughBook CF-08 is, as the name suggests, tough. It can withstand a 1.2 meter fall. The CF-08 is designed to be used via Citrix ICA or Microsoft RDP as a terminal server with either a ToughBook or a Windows server. The 10.4" is not only touch-capable, it is also readable in sunlight, a technology known as transflective LCD where the LCD has both transmissive and reflective properties. The all-magnesium case allows the CF-08 to be tough but also light. The CF-08 is sealed to IP54 specifications and has obtained other MIL-STD requirements.
