LG L206WU-PF Specifications

Display: 20" Wide TFT LCD
Aspect Ratio: 16:10
Pixel Format: 1680 x 1050
LCD: 8-bit
Response Time: 2ms
Brightness: 300 cd/m2
Contrast Ratio: 5000:1
Viewing Angle: 170/170
Connectivity: HDMI with HDCP, DVI-D with HDCP, VGA, USB 2.0
Position Adjustment: Tilt, Height, Swivel, Pivot
Speaker: Yes
Power Consumption: 32W (operational), 1W (standby)
Dimensions: 18.5 x 15.9 x 9.2"
Weight: 10.1lbs
Price: EUR 230

LG's L206WU-PF has a simple black design with a v-shaped area to indicate that the power is on: when it's on, it's blue. This is similar to the larger L246WP-BN. The response time is a fast 2ms and the dynamic contrast ratio is 5000:1. I've mentioned this in other posts but dynamic contrast is only useful when there are moving images. So for instance the combination of high dynamic contrast and a super fast response time should go really well together when you're playing fast-action games. Not only that, thanks to both HDMI and DVI you can watch HD content, even content that is protected. Of course the HD content will have to be scaled, which reduces the quality, but hey, this is a monitor not a TV. A boosted dynamic contrast would help when watching HD content too. The only limitation would be the viewing angle, thanks to the TN LCD panel, so when you're watching, make sure you're watching alone or in the middle.

Source: LG Japan via Akihabara News, ZDNet
