ZDNet:

Microsoft will push Windows Phone 7 software updates to end users and all Windows Phone 7 devices will be eligible for updates.

That makes sense. Carriers push Android updates and with so many different versions of Android and hardware configurations it is a mess. Apple, of course, pushes iOS updates to its iDevices and the process is simple and clean. Microsoft is using the second approach, which I agree that it should.
