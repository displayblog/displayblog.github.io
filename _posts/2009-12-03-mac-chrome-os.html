Google's "Mac OS X Roadmap" for its Chrome OS states:

Now that we have an initial beta release under our belts...


The Chrome OS is coming to the Mac very soon. If you would like to test drive a build there is a 20MB build 33770: chrome-mac.zip dated 03-Dec-2009 18:23. According to ZDNet the 33770 build "works quite well. It plays Flash, syncs your bookmarks with Google docs and scores 100/100 on the Acid 3 browser test."
