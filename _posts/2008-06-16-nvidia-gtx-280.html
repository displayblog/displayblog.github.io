NVIDIA GTX 280 Specifications
TPCs: 10
Streaming Multiprocessors (SM) per TPC: 3
Streaming Processors (SP) per SM: 8
Total SPs: 240 (up 87.5% compared to 8800 GTX)
TEX: 80t/clk (up 25%)
ROP Blend: 32p/clk (up 167%)
Precision: fp64 (from fp32)
GFLOPs: 933 (up 80%)
FB Bandwidth: 142GB (up 65%)
Texture Fill: 48 GT/s (up 29.7%)
ROP Blend: 19 GBL/s (up 171%)
PCI Express: 12.8 GB (up 100%)
Video: VP2 (from VP1)
Theoretical Bilinear Fillrate: 51,840 (up 54.3% compared to 9 Series)
Measured Rate (3DMark Multitex): 48,266 (up 88.5%)
Measured Performance / Theoretical Performance: 93.1% (up from 76.2%)
Availability: June 17, 2008 from ASUS, BFG, Emtek, EVGA, Gainward, Galaxy, Gigabyte, Innovision, Leadtek, MSI, Palit, PC Partner/Zotac, PNY, Point of View, Unika, Unitek, and XFX
Price: $649

Original Post

Watch out Intel. You too AMD. NVIDIA is playing hardball and with its latest GPU, the GTX 280. NVIDIA's got a rock-solid knock-out punch with its GTX 280 and below are some incredible details of what it can do.

CUDA Computing Mode: The CUDA general-purpose parallel processing mode has supercomputing power to simulate realistic, physically accurate effects. With CUDA runtime for Windows Vista, a lot of processing tasks can be offloaded from the CPU to a NVIDIA GPU.


Warrior from NVIDIA Medusa Demo. This looks so real, it looks like a picture to me!

According to Vijay Pande, Associate Professor of Chemistry at Stanford University, the GeForce GTX 280 is able to run Folding@Home much faster than the latest 3GHz Core 2 Quad CPU from Intel. 45 TIMES FASTER.

Here's the proof in the pudding according to NVIDIA:
146x faster than CPU for interactive visualization of volumetric white matter connectivity
36x: Ionic placement for molecular dynamics simulation
18x: Transcoding HD video stream to H.264 for portable video
17x: Simulation in Matlab using mex file CUDA function
100x: Astrophysics N-body simulation
149x: Financial simulation of LIBOR model with swaptions
47x: GLAME@lab M-script API for linear Albegra operations on GPU
20x: Ultrasound medical imaging for cancer diagnostics
24x: Highly optimized object oriented molecular dynamics
30x: Cmatch exact string matching - find similar proteins & gene sequences

2nd Generation NVIDIA Unified Architecture: 50% faster gaming performance compared to the GeForce 8800 Ultra. 240 enhanced processor cores that can pump out to 2560 x 1600 pixels.

3-way NVIDIA SLI Technology: 3-WAY. NVIDIA claims that its 3-way alternate frame rendering (AFR) as the world's fastest gaming solution under Windows Vista, I can believe it.

NVIDIA PureVideo HD Technology: HD video decoding is accelerated. Picture clarity, video smoothness, color accuracy, image precision are all enhanced.
